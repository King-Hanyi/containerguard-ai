**第二步：如何填写"实际结果"**

运行以下命令获取实际数据：

```bash
# SSH 到服务器
ssh root@你的服务器IP
cd /root/containerguard-ai

# 1. 运行 baseline 获取 CVE 判定结果
uv run python scripts/run_baseline.py
# 结果保存在 docs/baseline_results.json

# 2. 查看结果
cat docs/baseline_results.json | python3 -m json.tool

# 3. 对照 baseline_results.json 填写表格中的"实际判定"和"实际置信度"
```

每个 CVE 的结果在 JSON 中的格式：
```json
{
  "CVE-2021-44228": {
    "status": "affected",       ← 填到"实际判定"
    "confidence": 1.0,          ← 填到"实际置信度"
    "time_seconds": 1.48
  }
}
```

**第三步：与期望比对**

把每行的"实际判定"和"期望判定"对比，如果一致就在"通过"列写 `✅`，不一致写 `❌` 并在备注中说明原因。

## ⚠️ 注意事项

1. **先 `git pull` 再开始**: 项目最近更新频繁，确保拉取最新代码
2. **提交规范**: `git add -A && git commit -m "test: 描述" && git push origin main`
3. **截图方法**: 服务器终端可以直接复制文本粘贴到文档，浏览器可以截屏
4. **遇到问题**: 如果脚本报错，先截图记录错误信息，然后在飞书群找组长
5. **不需要写代码**: 你的任务主要是**运行现有脚本 + 记录结果 + 填写表格**
