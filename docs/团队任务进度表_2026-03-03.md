#### 📌 给卢周全的具体指引

**C4 是最重要的** — 因为你的 KnowledgeGraph 模块目前是孤立的，没人调用它。需要：
1. 在 `intel_agent.py` 中 `import KnowledgeGraph`
2. 在 Intel Agent 收集情报后，调用 `kg.query_attack_chain(cve_id)` 获取 CWE/CAPEC/ATT&CK
3. 把攻击链数据追加到 `IntelResult` 的 `intel_data` 字段中
4. 这样 VEX Agent 的 LLM 推理 prompt 会自动包含攻击链信息

**C5 建议用 `rank_bm25` 库**：
```bash
pip install rank-bm25
```
对所有 CVE 的 description 字段建 BM25 索引，支持关键词检索相似漏洞。
